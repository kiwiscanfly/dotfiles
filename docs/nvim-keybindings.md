# Neovim Keybindings

Leader key is `Space`.

## Finding Files (Telescope)

| Shortcut | Action |
|----------|--------|
| `<leader>ff` | Find files |
| `<leader>fg` | Grep in files (live search) |
| `<leader>fb` | Find buffers |
| `<leader>fh` | Help tags |

## LSP

| Shortcut | Action |
|----------|--------|
| `gd` | Go to definition |
| `gr` | Find references |
| `K` | Hover documentation |
| `<leader>ca` | Code actions |
| `<leader>rn` | Rename symbol |
| `<leader>x` | Show diagnostic float |

## Debugging (DAP)

| Shortcut | Action |
|----------|--------|
| `F5` | Start/Continue |
| `F10` | Step Over |
| `F11` | Step Into |
| `F12` | Step Out |
| `<leader>b` | Toggle breakpoint |
| `<leader>B` | Set conditional breakpoint |

## Flutter

These keybindings are only active in Dart files.

| Shortcut | Action |
|----------|--------|
| `<leader>fs` | Run Flutter app |
| `<leader>fr` | Hot Reload |
| `<leader>fR` | Hot Restart |
| `<leader>fq` | Quit Flutter |
| `<leader>fd` | Select device |
| `<leader>fe` | List emulators |
| `<leader>fo` | Toggle outline |
| `<leader>fc` | Clear log |
| `<leader>fv` | Switch FVM version |
| `<leader>fD` | Detach |
| `<leader>fl` | Restart LSP |

## Which-Key Groups

Press `<leader>` and wait to see available groups:

| Prefix | Group |
|--------|-------|
| `<leader>f` | Find |
| `<leader>g` | Git |
| `<leader>l` | LSP |
| `<leader>b` | Buffer |
| `<leader>d` | Debug |
| `<leader>t` | Test |

## Standard Vim

These are not custom but commonly used:

| Shortcut | Action |
|----------|--------|
| `u` | Undo |
| `Ctrl+r` | Redo |
| `/` | Search forward |
| `?` | Search backward |
| `n` / `N` | Next/previous search result |
| `*` | Search word under cursor |
| `gg` / `G` | Go to top/bottom |
| `zz` | Center cursor on screen |
| `Ctrl+o` / `Ctrl+i` | Jump back/forward |
