# Hyprland Keybindings

Mod key is `Super` (Windows/Command key).

## Essential Controls

| Shortcut | Action |
|----------|--------|
| `Super+Return` | Open terminal (Ghostty) |
| `Super+Q` | Close focused window |
| `Super+Escape` | Lock screen |
| `Super+Space` | Application launcher (Wofi) |
| `Super+E` | File explorer (Nautilus) |
| `Super+Shift+C` | Reload config |
| `Super+Shift+E` | Exit Hyprland |

## Navigation (Vim-style)

| Shortcut | Action |
|----------|--------|
| `Super+H` | Focus left |
| `Super+J` | Focus down |
| `Super+K` | Focus up |
| `Super+L` | Focus right |

Arrow keys also work: `Super+Left/Down/Up/Right`

## Moving Windows

| Shortcut | Action |
|----------|--------|
| `Super+Shift+H` | Move window left |
| `Super+Shift+J` | Move window down |
| `Super+Shift+K` | Move window up |
| `Super+Shift+L` | Move window right |

Arrow keys also work: `Super+Shift+Left/Down/Up/Right`

## Workspaces

| Shortcut | Action |
|----------|--------|
| `Super+1-9, 0` | Switch to workspace 1-10 |
| `Super+Shift+1-9, 0` | Move window to workspace 1-10 |

## Layout

| Shortcut | Action |
|----------|--------|
| `Super+B` | Preselect split left |
| `Super+V` | Preselect split down |
| `Super+S` | Toggle split direction |
| `Super+F` | Toggle fullscreen |
| `Super+Shift+Space` | Toggle floating |
| `Super+D` | Cycle floating windows |
| `Super+A` | Focus urgent or last |

## Scratchpad

| Shortcut | Action |
|----------|--------|
| `Super+Minus` | Toggle scratchpad |
| `Super+Shift+Minus` | Move window to scratchpad |

## Resize Mode

Press `Super+R` to enter resize mode:

| Key | Action |
|-----|--------|
| `H` / `Left` | Shrink width |
| `L` / `Right` | Expand width |
| `K` / `Up` | Shrink height |
| `J` / `Down` | Expand height |
| `Return` / `Escape` | Exit resize mode |

## Mouse

| Action | Effect |
|--------|--------|
| `Super+Left Click Drag` | Move window |
| `Super+Right Click Drag` | Resize window |

## Media Keys

| Key | Action |
|-----|--------|
| `XF86AudioMute` | Toggle mute |
| `XF86AudioLowerVolume` | Volume -5% |
| `XF86AudioRaiseVolume` | Volume +5% |
| `XF86AudioMicMute` | Toggle mic mute |

## Brightness

| Key | Action |
|-----|--------|
| `XF86MonBrightnessDown` | Brightness -5% |
| `XF86MonBrightnessUp` | Brightness +5% |

## Screenshots

| Shortcut | Action |
|----------|--------|
| `Print` | Screenshot full screen |
| `Shift+Print` | Screenshot selection |

Screenshots saved to `~/Pictures/Screenshots/`.

## System

| Shortcut | Action |
|----------|--------|
| `Super+N` | Toggle notification center |
| `Super+Shift+N` | Dismiss all notifications |
| `Super+Shift+P` | 1Password quick access |
